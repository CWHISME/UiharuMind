# UiharuMind 帮助

UiharuMind 是一个面向日常桌面使用的 AI 助手。它可以管理本地和远程模型，进行普通对话、翻译、截图识别、剪切板快捷处理、知识库对话，以及自动点击录制和回放。

## 快速开始

1. 打开“模型”页面，先添加一个可用模型。
2. 如果使用远程模型，添加兼容 OpenAI 接口的地址、模型名和 API Key。
3. 如果使用本地 GGUF 模型，先在“设置 -> 运行时”下载或选择本地执行后端，再在“设置 -> LLamaCpp 设置”和“模型”页面配置模型目录。
4. 回到“对话”页面，在顶部模型选择框中选择要使用的模型，然后开始输入。

如果模型没有显示，优先检查模型目录、远程模型配置、运行时是否已下载，以及当前模型文件是否为支持的格式。

## 模型管理

### 本地模型

本地模型主要面向 GGUF 量化模型。将模型文件放入模型目录后，应用会在“模型”页面中扫描并展示。

本地模型需要运行时后端支持。如果“运行时”列表为空，可以在“设置 -> 运行时”中检查更新并下载适合当前系统的后端。

本地模型运行会占用较多 CPU、内存或显存。模型越大，对机器配置要求越高。

### 远程模型

远程模型使用在线 API。只要服务兼容 OpenAI 接口，一般都可以通过“模型”页面添加。

API Key 会在本机加密保存。配置文件复制到另一台机器后，API Key 通常需要重新填写。

### 多模态模型

截图识别、视觉问答等功能需要支持图片输入的模型。如果当前模型不支持视觉能力，应用会尝试从已添加的远程模型中查找可用的视觉模型。

## 对话与角色

“对话”页面用于普通聊天和角色对话。你可以创建不同对话，切换模型，使用角色设定，并查看历史消息。

对话内容支持 Markdown 渲染。如果想查看原始 Markdown 文本，可以在设置中调整显示相关选项。

右键对话列表或消息头像，可以进行删除、编辑等操作。快捷询问产生的临时对话会同步到主界面对话历史中，但通常只保留最近一次；如果需要长期保存，可以复制一份。

## 知识库对话

在对话输入区域上方可以管理当前对话绑定的知识库。知识库可以包含文件、目录或 URL，用于让模型回答时参考指定资料。

使用知识库前，建议先确认资料内容准确、文件格式可读取，并选择适合长文本检索和总结的模型。

## 快捷功能

全局快捷键可以在“设置 -> 快捷键”中自定义。默认快捷键包括：

| 功能 | 默认快捷键 |
| --- | --- |
| 截图 | Alt+Shift+Z |
| 快捷询问 | Alt+Shift+A |
| 剪切板历史 | Alt+Shift+S |
| 翻译 | Alt+Shift+Q |
| 自动点击工具 | Alt+Shift+G |

### 截图

截图后可以进行 OCR、视觉问答、解释图片、编辑、复制等操作。截图内容被编辑后，工具栏会提供切换原图和编辑图的入口。

在 macOS 上，截图和全局快捷键依赖系统权限。如果重新安装后权限异常，可以在系统设置中移除旧权限记录，再重新打开应用授权。

### 快捷询问

复制文本后，可以通过快捷询问快速把当前内容交给模型处理。应用会优先使用当前运行模型；如果没有手动加载模型，会尝试从已添加的远程模型中选择可用项。

### 剪切板历史

剪切板历史用于查看最近复制过的文本内容。复制结果可以再次选择、发送给模型或用于其它快捷操作。

### 翻译

翻译窗口用于快速翻译剪切板内容或手动输入内容。翻译使用当前模型或已配置的远程模型，并可结合翻译角色设置调整风格。

### 自动点击

自动点击工具可以录制鼠标、键盘、滚轮动作，并按录制顺序回放。录制和回放时可以使用设置中的自动点击快捷键快速打开或停止。

Windows 游戏输入支持普通权限、无反作弊拦截的场景。如果目标游戏以管理员权限运行，UiharuMind 也需要以同等权限运行。应用不提供驱动级、注入或绕过反作弊的方案。

## 设置

设置窗口集中管理应用行为：

- “通用”：语言、应用更新、日志和常用应用设置。
- “运行时”：本地模型执行后端的下载和选择。
- “LLamaCpp 设置”：本地模型服务参数、采样参数和调试配置。
- “快捷键”：修改截图、快捷询问、剪切板历史、翻译和自动点击快捷键。
- “帮助”：查看当前文档。
- “关于”：版本信息、项目链接和依赖信息。

语言切换会立即应用到主要界面。少量已经打开的复杂窗口如果没有完全刷新，关闭后重新打开会使用新的语言。

## 常见问题

### 模型目录中有文件，但模型列表没有显示

确认文件是支持的 GGUF 模型，模型目录配置正确，并尝试重新打开应用。如果是第一次配置本地模型，还需要先在“设置 -> 运行时”中准备本地执行后端。

### 远程模型无法调用

检查 API 地址、模型名、API Key 和网络连接。确认服务兼容 OpenAI 接口，并且当前账号有调用权限。

### macOS 提示需要辅助功能权限

打开系统设置中的辅助功能权限，允许 UiharuMind 控制电脑。如果权限列表里已有旧记录但仍无效，删除旧记录后重新打开应用并再次授权。

### Windows 游戏内快捷键或回放无效

确认游戏没有启用反作弊拦截，且应用和游戏权限等级一致。如果游戏以管理员权限运行，UiharuMind 也需要以管理员权限运行。

### 修改快捷键后菜单显示没有变化

快捷键保存后，托盘菜单中的快捷键提示会随配置刷新。如果仍显示旧内容，可以关闭并重新打开应用确认配置是否已保存。
